# Warped Pixel Dungeon

**Warped Pixel Dungeon** is an open-source roguelike dungeon crawler built on top of [Shattered Pixel Dungeon](https://shatteredpixel.com/shatteredpd/) by [Evan Debenham](https://github.com/00-Evan), which itself is based on [Pixel Dungeon](https://github.com/00-Evan/pixel-dungeon-gradle) by [Watabou](https://watabou.itch.io/).

It incorporates ideas and code from several other PD mods including [Sprouted PD](https://github.com/dachhack/SproutedPixelDungeon), Overgrown PD, and others — blending them into a unique experience with new mechanics, items, and systems.

## What's Different?

- Temperature & weather system (hypothermia, heatstroke, seasons)
- Passive plant growth and grass regrowth cycles
- Expanded inventory (43 usable backpack slots)
- Food pouch, Ankh chain, Key ring bags
- Living plants that grow from seeds
- New potions, items, and mobs ported from multiple PD mods
- And more weird stuff in the works...
